============================================
    GTA V Script Cam Tool by CamxxCore
============================================

--- Prerequisites ---

ScriptHook V - http://www.dev-c.com/gtav/scripthookv/

ScriptHook V .NET - https://github.com/crosire/scripthookvdotnet/releases/tag/v1.1

--- Installation ---

1. Create a folder called "scripts" in the main folder where GTA is installed

2. Place GTAV_ScriptCamTool.dll in the folder
